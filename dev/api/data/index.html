Reversi.GameRecord — Type
GameRecord

Stores the complete move history of a finished (or in-progress) Reversi game as a vector of standard notation strings (e.g. ["d3", "c5", ...]).

Fields:

  • moves::Vector{String} – ordered list of moves in standard notation.
  • result::Int – BLACK, WHITE, or EMPTY (draw); -1 when not yet finished.

Reversi.play_game — Method
play_game(player1::Player, player2::Player;
          verbose::Bool=true, save_record::Bool=false,
          record_path::String="game_record.txt") -> Int

Extended version of play_game that optionally saves the move history. When save_record=true the record is written to record_path after the game. Returns the winner (BLACK, WHITE, or EMPTY).

Reversi.replay_game — Method
replay_game(record::GameRecord; verbose::Bool=false) -> ReversiGame

Replay all moves stored in record on a fresh ReversiGame and return the resulting game state. Passes are represented by the special token "pass".
